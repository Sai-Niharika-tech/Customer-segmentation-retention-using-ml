# Customer Segmentation and Churn Prediction

## Project Overview
This project analyzes customer behavior to identify different customer segments and predict customer churn using machine learning techniques.

## Objectives
- Segment customers using K-Means clustering
- Predict churn using Logistic Regression
- Analyze churn patterns across customer segments

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Google Colab

## Results
The project successfully identified customer segments and churn-prone customers. 
Visual analysis was used to derive customer retention insights.
